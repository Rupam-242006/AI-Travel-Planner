# 🌍 Wandr - AI Travel Planner

📌 **Pictures**

Wandr is an AI-powered travel planner that helps users create personalized trips according to their interests, preferences, and budget.

It simplifies travel planning by suggesting destinations, estimating expenses, and organizing trip ideas in a smarter way.

---

## ✨ Features

📌 **AI-Based Trip Planning**
- Generates travel plans based on user interests.

📌 **Expense Estimation**
- Helps users plan trips according to their budget.

📌 **Personalized Recommendations**
- Suggests travel ideas based on preferences.

📌 **Simple & User-Friendly Interface**
- Clean design for an easy planning experience.
